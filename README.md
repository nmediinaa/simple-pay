# 💰 Simple Pay API

Este projeto consiste no desenvolvimento de uma API para uma carteira digital simplificada. O objetivo principal é consolidar conceitos avançados de arquitetura de software, segurança, persistência e regras de negócio utilizando o ecossistema .NET.

A aplicação gerencia usuários, permitindo que realizem transferências de dinheiro entre contas, respeitando regras de saldo e permissões específicas.

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** C# (.NET 8)
* **Framework Principal:** ASP.NET Core Web API
* **ORM / Persistência:** Entity Framework Core
* **Banco de Dados:** MySQL
* **Design Patterns & Boas Práticas:** Padrão DTO (Data Transfer Objects), Injeção de Dependência e Programação Orientada a Objetos (POO).

## 🚧 Status do Projeto & Roadmap (Em Desenvolvimento)

O projeto está em fase ativa de desenvolvimento.

## ⚙️ Como Executar o Ambiente Local

### Pré-requisitos
